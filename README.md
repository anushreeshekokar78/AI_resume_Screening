# 🧠 AI Resume Screening System

## 📌 Overview

The **AI Resume Screening System** is a full-stack web application designed to automate the process of analyzing and ranking resumes based on job requirements. It helps recruiters quickly shortlist candidates using intelligent keyword matching and scoring.

---

## 🚀 Features

* 📄 Upload resumes (PDF/DOCX)
* 🔍 Extract text from resumes
* 🧠 Analyze keywords using NLP
* 📊 Score resumes based on job description
* 📋 Display ranked candidates
* ⚡ Fast and efficient screening process

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/anushreeshekokar78/AI_resume_Screening.git
cd AI_resume_Screening
```

---

### 2️⃣ Backend Setup (Django)

```
cd ai-resume-screening
python -m venv venv
venv\Scripts\activate
pip install -r ../requirements.txt
python manage.py runserver
```

---

### 3️⃣ Frontend Setup (React)

```
cd ai-resume-frontend
npm install
npm start
```

---

## 📊 How It Works

1. User uploads resume
2. Backend extracts text
3. NLP processes keywords
4. Resume is scored based on relevance
5. Results displayed on frontend
